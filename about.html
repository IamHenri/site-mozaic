Objectifs

Mozaic vous permet d'organiser vos contenus visuellement, trouver et accéder à une ressource en un clic, la partager en deux.

Génèse du projet

Mozaic est née d'un besoin : disposer des marque-pages depuis n'importe où, quelque soit le terminal informatique auquel on accède, où le lieu.

Le véhicule le plus naturel pour bénéficier d'un tel service est une page web unique avec tout présenté sous forme d'icônes rangées dans des dossiers et sous dossiers, accessibles en un clic.

Code de conduite

Mettre à disposition un portail d'entrée à la navigation sur le web pour tous les internautes devant accéder à de multiples ressources sur web en respectant :

  • les utilisateurs, seul un email est nécessaire pour se connecter;
  • leurs données, les marque-pages de tous les utilisateurs sont leur propriété exclusive;
  • l'email, s'il est utilisé, il le sera uniquement pour des communications nécessaires relatives à Mozaic.
  • la version gratuite dans sa version le restera.
